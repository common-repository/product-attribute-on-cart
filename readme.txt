== Installation ==
1. Upload the folder "product-attribute-on-cart" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, product attribute will automatically add on cart page.
4. Enjoy!
